# Student Grades Calculator (C++)

This application calculates the final grades of students based on homework and exam results.
Users can choose between **average** or **median** as the final grade calculation method.

### Formula for average-based final grade:
**Final Grade = 0.4 × Homework Average + 0.6 × Exam Score**

Future updates:
- Median-based grading
- Random data generation
- File input/output
